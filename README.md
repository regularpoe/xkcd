# xkcd

```
USAGE:
    xkcd [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -n, --number <number>    Comic number to fetch [default: 0]
    -o, --output <output>    Print output in format: text/json [default: text]
    -s, --save <save>        Save image to current directory
```
